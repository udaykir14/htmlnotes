LISTS:

list is a group of items.
we have 3 types of lists.

  1. ordered list
  2. unordered list
  3. definition list

ORDERED LIST

  • to create order list we use < ol >and < li > tags.
  • we use type and start attributes here.
  • type attribute is used to change the type of ordered list
  • we can change type to i,I,1,a,A.
  • start attribute is used to give starting number for the first list item in order list.
  • syntax:< ol >
    < li > -----< /li >
    < /ol >

UNORDERED LIST

  • to create unordered list we use < ul >and < li > tags.
  • we use type attribute here.
  • type attribute is used to change the type of unordered list
  • we can change type values are disk,circle,square,none .
  • syntax:< ul >
    < li > -----< /li >
    < /ul >

DEFINITION LIST

  • to create defenition list we use < dt > , < dd > and < dl > tags.
  • dt-data title
  • dd-data description
  • dl-data list
  • syntax:< dl >
    < dt > -----< /dt >
    < dd > -----< /dd >
    < /dl>
