FORMS

  • Forms are used to collect the information from the users.

  • The submited form will be sent to the backend in the form of name and value pair(name='value')

  • To create forms we use form tag < from > which is paired and block level element.

ATTRIBUTES OF FORM TAG

METHOD

ACTION

  • Here we can use https methods like get and post.
  • Get method is default ,here submited data is visible in the url.
  • Post method can be used to unvisible the data in the url.
  • Post is secure method.
