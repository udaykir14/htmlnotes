TABLES

  • In html we use tables to display data in the form of rows and columns.
  • To create a table we use table tag which is a paired tag.
  • Inside table tag we use < thead > , < tbody > ,< tfoot > tags.
  • These tags describes table head, table body,table footer'
  • To enterr data in a table we create a row then we use < td > or < th > tags.
  • In the table we use < tr > tag to create a row.
  • < th > tag display the data in bold font and its content will be exactly at the center of the cell.
  • th tag is used in thead .
  • < td > tag is used to display the data of the table

ATTRIBUTES IN TABLE TAG
BORDER

It is used to give border for the table.
HEIGHT

It is used to give height fro the table.
WIDTH

It is used to give the width for the table.
CELLSPACING

It is used to set space between the cells.
CELLPADDING

It is used to set the space between the content and the table margin.
ALIGN

It is used to align the table to center ,left or right.




ATTRIBUTES IN < TH >,< TB > TAGS
ROWSPAN

It is used to merge two or more cells present in adjacent rows.
COLSPAN

It is used to merge two or more cells present in adjacent columns.
