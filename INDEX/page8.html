MEDIA TAGS:

  • We use media tags to display image,audio,video in the webpage.


1.IMAGE TAG 2.AUDIO TAG 3.VIDEO TAG

To display image in the webpage we use < img > tag,

it is inline and unpaired tag.

To display audio in the webpage we use < audio > tag,

it is inline and paired tag.

To display image in the webpage we use < video > tag,

it is inline and paired tag.

Attributes of img tag are src(source),height,width,

alt(alternate meaasge)

Attributes of audio tag are src(source),controls

Attributes of video tag are src(source),autoplay,controls,loop,muted.

src------>used to mention the path of img.

height-->used to set the height of the image.

width--->used to set width of the image.

alt------->We can give alternate msg when browser cant reach the path.

src--->used to mention the path of audio.

controls-->are mandetory attribute .

src--->used to mention the path of video.
control--> are mandetory attribute.

autoplay-->used to play video automatically in the browser.

loop--->used to play video repeatedly.

muted-->sound of the video gets muted by using this attribute.

syntax: < img src=" " height=" " widht=" " alt="" >

syntax: < audio src=" " controls >

syntax: < video src=" " controls muted loop autoplay
