TAGS

  • Tags are predefined words.
  • content is written within ' < ' and ' > ' braces is called a tag.
  • Tags are used to create elements.

Types of Elements
block level elements: bolck which occupies entire widht of the content is called block level elements
In line level elements: block which occupies only width of the content is called in line level element


Types of tags
paired tags:

Tags which contains both opening and closed tags are called as paired tags
unpaired tags:

Tags which contains only opening tags are called as unpaired tags

Preformated tag
<pre> tag</pre>

it is similar to < p > tag, but it displays empty spaces and line brakes.

<strong>

<sub>

it is used to display content in the subscript form as chemical formulaes

it is used to highlight the content by underlining it.
