TYPES OF TAGS IN FORMS:

LABEL:

  • it is used to give the name for the input tag.

  • it is a paired tag and inline level element.

  • we use 'for' attribute inside the label tag.

  • for is used for reference of input tag

SYNTAX: < label for =" ">content< /label >

INPUT:

  • To create input field we use input tag < input > , user enters the data in it.

  • It is unpaired and inline level element.

  • Attributes in input tag are
    • Type
      • Text
      • Number
      • Email
      • password
      • submit
      • radio
      • checkbox
      • date
      • file
      • color
      • range
      • reset
    • Name
    • Value
    • Placeholder
    • Required
    • min
    • max
    • min length
    • max length
    • checked
    • autofocus
    • autocomplete

SYNTAX: < input type =" " name=" " value=" " placeholder requried min max>content
