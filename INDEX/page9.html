MARQUE TAG < marque >:

  • This tag is use to scroll the content on the webpage by default
    it will scroll from right to left.

  • It is a paired tag.

ATTRIBUTES :

DIRECTION:

SCROLLAMOUNT:

BEHAVIOUR:

  • It is used to change the direction of the scroll content.
  • Default value is left, it can be changed to right,up,down.

  • It is used to increase or decrease the speed of the scrolling contant.

  • It is used to change the behaviour of the marqur tag.
  • Default value is scroll, we can change it to alternate, slide.

Syntax: < marque behavoiur=" " direction=" " scrollamount=" "> content< /marque >
