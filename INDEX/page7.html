HYPERLINKS:

  • To create hyperlinks we use anchor tag (< a >) ,which is a paired tag and inline level element.
  • Hyperlinks are used to navigate from one source to another source.
  • There are two of hyper links.

INTERNAL HYPERLINK:

Internalhyperlinks are used to navigate with in the same webpage.

EXTERNAL HYPERLINK:

External hyperlinks are used to navigate from one webpage to other.



Hyperlinks has 3 states.
1.UNVISITED

By default unvisited link will appear in blue colour.
2.VISITED

By default visited link will appear in purple colour.
3.ACTIVE

By default active link will appear in red colour.


ANCHOR TAG (< a >):

It has 2 attributes

1.HREF

  • Hypereference is used to give path(url).
  • We can use absolute or relative url
  • absolute url is internet pathe.
  • relative url is file path.

2.TARGET

  • It is used to specify where to open the link document either in same tab or new tav.
  • Values for target attribute are '_parent', '_child', '_drop', '_self'.
