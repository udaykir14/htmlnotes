OTHER FORM TAGS:

TEXTAREA:

  • It is used to get feedback or comment box

  • Attributes are:
    • Name
    • rows
    • cols

SYNTAX: < textarea name=" " rows=" " cols=" ">content< /TEXTAREA >


FIELDSET:

  • It is used to group the related data in the form.

  • It gives border for the grouped data.

SYNTAX:< fieldset > content < /fieldset >.

LEGEND:

  • We use legend tag to give name for the fieldset.

SYNTAX:< legend > content < /legend >.

SELECT AND OPTION:

  • They are used to create dropdown lists.

SYNTAX:< select name=" " > content < option name=" " value=" " >content< /option >< /select >.
